\# Speech Preprocessing with LibriSpeech



This project implements a complete speech preprocessing pipeline using the LibriSpeech test-clean subset.



\## Features

\- Audio loading

\- Resampling to 16 kHz

\- Normalization

\- Silence trimming

\- Filtering

\- Waveform visualization

\- Mel spectrogram extraction

\- Feature tensor creation



\## Dataset

LibriSpeech test-clean subset (downloaded separately)



\## Requirements

\- Python 3.8+

\- librosa

\- numpy

\- matplotlib

\- torch



\## Run

```bash

python "speech preprocessing pipeline.py"
